# World Model Lab

> 世界模型实验室：一种用 AI 共同推演文明、历史与现实的学习协议。

World Model Lab 不是一门历史课，也不是一份哲学教材。

它是一个 **prompt-first** 的开源学习项目，目标是帮助人们通过长期对话，建立自己对世界如何运行的理解框架。

我们不追求一次性得到标准答案，而是追求：

- 更好的问题
- 更清晰的概念
- 更可复用的框架
- 更强的独立判断力

---

## 这个项目是什么？

World Model Lab 是一个与 AI 共同思考的提示词协议。

它鼓励你把对话当成一场“文明模拟器”：

```text
合作如何出现？
权力从哪里来？
为什么会有不平等？
秩序为什么会腐化？
为什么文明会循环，但人类整体又在进步？
故事、宗教、法律、货币、国家、公司、AI，本质上解决了什么问题？
```

我们不是把知识存给 AI，而是用 AI 帮助自己训练判断力。

---

## 这个项目不是什么？

它不是：

- 百科知识库
- 历史时间线
- 哲学鸡汤
- 提示词技巧合集
- AI 自动总结工具

它更像是一种思考协议：

```text
模拟场景
→ 做出判断
→ 暴露假设
→ 形成概念
→ 比较分支
→ 修正模型
→ 继续推演
```

## 快速开始

打开你常用的 AI 对话工具，复制 [`PROMPT.md`](./PROMPT.md) 的内容，然后开始对话。

推荐第一个问题：

```text
为什么人类会合作？
```

不要让 AI 直接给你答案。

让它先构造一个原始人类小群体场景，然后问你：

```text
如果你在那个时代，你会设计什么规则，让这个群体长期合作下去？
```

你的回答才是实验的起点。

## 核心方法

### 1. 不从学科开始，从问题开始

不要先问：

```text
经济学是什么？
政治学是什么？
历史学是什么？
```

而是问：

```text
为什么陌生人愿意合作？
为什么弱者服从强者？
为什么强者也需要合法性？
为什么秩序会保护既得利益者，也保护普通人？
为什么简单故事比复杂真相更容易传播？
```

---

### 2. 不追求答案，追求模型

一次对话得到的结论不重要。

重要的是把结论标记成：

- 事实
- 推论
- 假设
- 信念
- 开放问题

这样后续才能不断修正。

---

### 3. 把历史当成模拟器

历史不是用来背诵的。

历史是用来训练判断的。

在每一个关键节点，我们都可以暂停，然后问：

```text
如果我是当时的部落首领、皇帝、商人、农民、工程师、革命者、创业者，我会怎么选？
```

选择本身不重要。

选择背后的世界模型才重要。

## 已出现的世界模型规则

### Rule #001

小规模合作靠认识，大规模合作靠相信。

### Rule #002

竞争驱动组织追求效率，效率会自然产生不平等，不平等积累到一定程度会削弱合法性，合法性下降又会反过来削弱效率。

### Rule #003

只要存在差异，就会产生不公平感。

### Rule #004

秩序不是自然状态，而是持续投入能量对抗熵增的结果。

### Rule #005

文明会重置，知识会累积。

## 基本信念

World Model Lab 暂时基于几个工作假设：

1. 世界不是由单一因素驱动的，而是由合作、激励、资源、权力、信息、故事、技术等因素共同演化。
2. 好的世界模型应该同时解释历史循环和长期进步。
3. AI 最有价值的用途之一，不是替人保存知识，而是帮助人训练判断。
4. 真正的学习不是“我听懂了”，而是“我能在复杂局面中做出更好的判断”。

这些都不是最终答案，只是 v0.1 的起点。
